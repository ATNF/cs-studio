CSS Introduction

The Graphical User Interface of the Control System Studio is based upon the basic Eclipse RCP UI components:


Views
A view is an UI element that typically provides a “view” on an underlying data model. The Control System Studio provides various views that can be easily arranged by changing their locations on the screen. Views can be positioned on various locations.

 

Editors
An editor is typically used to “edit” an underlying data model. The conceptual main difference between views and editors is that an editor provides only perform changes in the data model, when it is explicitly saved. There is only one place on the screen, where editors can appear (editor area).

 

Perspectives
Generally, a perspective defines an arrangement of various views. Also certain menu or toolbar entries can be associated to a perspective.
