File Appender Settings

 

Option

Description

Default

Log level

Der Log-Level gibt die Schwelle für Log-Nacrichten an, die zum File-System geschrieben werden. Es werden nur Log-Nachrichten geschrieben, die mindestens den gewählten Log-Level haben. Mögliche Log-Level sind: INFO, DEBUG, WARNING, ERROR, FATAL.

INFO

Pattern

Dies Pattern bestimmt das allgemeine Layout von Datei-Log-Nachrichten, die von CSS erzeugt werden. Für eine Beschreibung der Pattern-Symbole bitte hier klicken.

%d{ISO8601} %-5p [%t] %c: %m%n

Log file

Die Ziel-Datei, auf die Log-Nachrichten geschrieben werden. Dieser Pfad ist relativ zum CSS-Installationsverzeichnis orientiert.

logs/event.log

Backup index

Wenn die Log-Datei die Größe von 500KB überschreitet, wird eine neue Log-Datei angelegt und die alte Datei wird archiviert, indem ein fortlaufender Backup-Index angehängt wird. Hier wird festgelegt, wieviele Backup-Dateien aufgehoben werden.
