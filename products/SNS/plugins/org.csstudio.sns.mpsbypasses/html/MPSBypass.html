MPS Bypass Table

The CSS MPS Bypass Table is a tool for tracking the MPS bypass states and MPS bypass request statuses from selected SNS Machine Modes. The tool reads the MPS Bypass information and Bypass Request status from the RDB.

The MPS Bypass model grabs all the RDB information for the input Machine Mode and stores the information in an array. MPS bypass PVs are created from RDB information as explained in the Overview. A second array is created, containing only the bypass information pertaining to the input bypass state, obtained from the created pvs, and bypass request status, obtained from the RDB. This array contains the data that is displayed in table on the GUI.
