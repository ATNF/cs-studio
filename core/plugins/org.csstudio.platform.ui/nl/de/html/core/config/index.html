CSS Core Configuration

Das Control System Studio wird mittels eines zentralen 'Preferences'-Dialogs konfilguriert, der mit dem “Preferences…” Menu-Item aus dem CSS-Menu geöffnet werden kann. Auf jeder 'Preference'-Seite können Einstellungen für entsprechende CSS-Core-Funktionen getätigt werden. Und auf jeder 'Preference'-Seite können die Einstellungen auf System-Default-Werte zurückgesetzt werden. Man kann nach 'Preference'-Seiten suchen, indem man Suchausdrücke in ein Filter-Feld eingibt.

 

Die verfügbaren Preference-Seiten sind in folgenden Kategorien organisieret:

 

  • Help
  • Help Server
  • Install/Update
  • Locale Settings
  • Logging
  • Console Appender
  • File Appender
  • JMS Appender
