Data Browser Changelog

Version numbers in here refer to the plugin org.csstudio.trends.databrowser2.

Version 3.1.1 - 2012-??-??

  • databrowser - Enhancement to show specified elements of waveform records in the chart.
  • databrowser - Changed to allow multiple items referring to the same PV to exist in one chart.

Version 3.1.0 - 2012-01-30

  • databrowser - Search with empty channel name pattern gives info dialog, no channels.
  • databrowser - New option to always use 'default' archive data sources, ignoring sources from (old) *.plt.
  • databrowser - New option to log archive access errors instead of prompting in dialog.

Version 3.0.2 - 2011-08-25

  • swt.xygraph - Fix for horizontal zoom that would 'invert' traces
