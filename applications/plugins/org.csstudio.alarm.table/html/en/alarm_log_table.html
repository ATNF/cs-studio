Alarm / Log Table

The Alarm / Log Table application receives messages of different kinds and displays them in table views.
There are three types of table views:

Log Table: The Log Table displays all types of received messages. They are ordered chronologically.

Alarm Table: The Alarm Table shows only alarm messages from the control system (Type 'event'). They are ordered firstly by severity and secondly by timestamp. In Alarm Table messages can be acknowledged.

Archive Table: The Archive Table queries the database for messages for a time period set by the user. A filter can be defined to get just messages with a certain properties.
