Changing Limit Values

Only limits that are displayed in "rw" columns can be changed. These acess permissions are defined in Configuration Files. If the tooltip for the pv limit value you want to change displays ReadWrite, the limits can be edited. See Tooltip Display for instructions on how to display tooltips.

Change A Single Limit Value

  • Click the left mouse button over the editable limit Tooltip Display.
  • Type the new limit value in the highlighted cell and press the Enter key. If pressing the left mouse button has no effect on the cell, you are trying to edit an noneditable limit.
  • Edited limit values are displayed with a yellow cell background and the Pace Editor title bar contains an * when the table has been edited. As seen in the Tooltip Display image.

Change Multiple Limits To The Same Value

To change multiple limits to the same value

  • Highlight the rows that contain the limits you want to change.
  • Press the left mouse button on the column header of the limits you want to edit. The screen below will be displayed, containing the column header you selected.

  • Enter the value you want the limits changed to.
  • Press the Enter key or the OK button to set the new value. The Input Value for Selected Rows window will disappear and within seconds the new values will be displayed in the selected column for the highlighted rows.
  • Or press the Cancel button on the Input Value for Selected Rows window to abort the transaction.
  • Edited limit values are displayed with a yellow cell background and the Pace Editor title bar contains an * when the table has been edited. As seen in the Tooltip Display image.
