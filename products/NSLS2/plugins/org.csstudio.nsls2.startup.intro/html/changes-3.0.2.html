Control System Studio for NSLS-II
3.0.2 Changes

Core

  • Reorganized menus to be more standard. Note: Preferences is now under Edit.

Probe

  • Improved visuals - new icon, meter stretches better.
  • Improved performance - doesn't queue events if pv updates fast, faster refresh.
  • Can open more then one instance.
  • Editor text does not update while entering a new value.

Channel Viewer (was Channel Finder)

  • New icon.
  • Remembers the last 10 queries.
  • Can open more then one instance.
  • Does not require authentication for reads.

Waterfall

  • New icon.
  • Provides an OPI widget.
  • Accepts channel queries.
  • Can order channels of a channel query by property.

PVTable by Property

  • New application.
  • Allows to organize the values from a channel query by using the channel properties.
  • Provides an OPI widget.
