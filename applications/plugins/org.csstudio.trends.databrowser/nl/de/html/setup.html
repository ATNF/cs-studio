Data Browser Perspective

Speziell für neue Benutzer empfiehlt sich das Umschalten auf eine voreingestellte Datenbrowser-Perspektive, die eine vordefinierte Anordnung des Plot-Windows, sowie einiger ähnlicher Views (Navigator, Konfiguraion, Archiv und Export) zur Verfügung stellt.
Aktivieren des Plot-Kontext-Menus mit Rechtsklick in den Datenbrowser-Plot, dann Auswählen des Menu-Eintrags

.

Wenn sich der aktuelle Schirm von dem hier unten dargestellten Schnappschuss signifikant unterscheidet, dann hat schon irgendjemand die Datenbrowser-Perspektive angepasst. Um diese Anpassungen rückgängig zu machen, Kontextmenu der Perspektive auswählen (Rechtsklick auf den Datenbrowser-Tab) und "Reset" auswählen.

Stopping a Data Browser Instance

Man kann ein Datenbrowser-Plot-Window einfach schliessen entweder über das übliche Symbol in der Window-Titelleiste oder über das Kontextmenu der Titelleiste.

Wenn die Datenbrowser-Konfiguration nicht gesichert wurde, entweder weil sie noch nie gesichert wurde, oder weil sie nach der letzten Sicherung geändert wurde, öffnet sich eine Dialogbox, in welchern man die Konfiguration sichern kann oder sie ohne sichern schliessen kann.

The Workspace, "Project" Folders

Konfigurationsdateien werden im Verzeichnis "Projects" im "Workspace" gesichert. Die CSS-Anwendung legt für den Benutzer ein Start-CSS-Projektverzeichnis an. Organisieren der Dateistruktur zur Erzeugung weiterer Projekte:
  1. Wenn es nicht sowieso schon geöffnet ist, erzeugen eines "Navigator"-View, zugreifbar über das Menu 
    Window/Show View.../GeneralNavigator
  2. Jedes top level Verzeichnis in der view ist ein "Projekt" im "Workspace". Erzeugen neuer Projekte über das Navigator-Kontextmenu 
    New Project
    Im erscheinenden Dialog "New Project" einen Projektnamen eingeben.

Saving Data Browser Configurations

Konfigurationsdateien werden in "Projekt"-Verzeichnissen gesichert. siehe im vorigen Abschnitt. Um Dateien zu sichern, kann man das Menu "File/Save As.." benutzen. Wenn die Konfiguration schon gesichert wurde, steht der aktuelle Konfigurationsdateiname in der Plow-Window-Titelleiste und der Eintrag "File/Save" im Menu erlaubt das Sichern von Änderungen in der aktuellen Konfigurationsdatei.

Re-loading a Data Browser Configurations

Man öffnet eine gesicherte Datenbrowser-Konfiguration durch Doppelklick auf die Konfigurationsdatei im "Navigator"

Creating a new, empty Data Browser Configurations

Man kann natürlich ein neues Datenbrowser-Plot-Window öffnen und dann, wie oben beschrieben, die Konfiguration mit "File/Save As" sichern.

Andererseits kann man eine neue, leere Datenbrowser-Konfigurationsdatei so erzeugen:

  • Öffnen der "Navigator"-View mit Menu 
    Window/Show View.../GeneralNavigator
  • Aktivieren des Kontextmenu 
    New/Other../CSS/Data Browser
  • Im der erscheinenden Dialogbox das Projektverzeichnis und den Konfigurationsdateinamen auswählen.
