PV Adjust, Confirm, Elog Tool

The PACE Editor provides a tabular view of PV names and their current limit values. This tool is to be used for changing and restoring limits.

In addition, when the limits changes are saved, an entry of the names of the pvs (EPICs Process Variables), the original limit values and the changed values are sent to the e-log.
