Adding PVs

There are several ways of adding Process Variables (PVs) to the Data Browser Plot.
  • Invoke the Plot's context menu, select "Add PV", enter the PV name.
  • Enter new names or change existing PV names in the Data Browser Config View.
  • Drop PV names from another CSS application into the Plot Window or the Config View.
  • In other CSS applications, select the desired PV names and use their context menu to send the PVs to a newly created Data Browser Plot.
