Setup

Suggested installation:
  • Start Matlab, then locate its installation direcory by tying this inside Matlab:
    matlabroot
  • Create a subdirectory 'arch' in there, and move the files in this package into that directory.
  • Either type the following each time you start matlab, or add it to the matlab startup.m file: 
    	javaaddpath(fullfile(matlabroot, 'arch', 'channelarchiver.jar'))
	addpath(fullfile(matlabroot, 'arch'))

    Search the Matlab online help for 'startup.m' in case you want this to happen automatically on startup.

Usage

Read arch_demo1.m, arch_demo2.m and get_archived_doubles.m.
