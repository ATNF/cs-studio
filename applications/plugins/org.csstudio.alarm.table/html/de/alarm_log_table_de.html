Alarm / Log Table

Die Alarm / Log Table Anwendung empfängt Nachrichten verschiedener Art und stellt sie in table views dar.
Es gibt 3 Arten von table views:

Log Table: Die Log Table stellt alle Typen der empfangenen Nachrichten dar. Sie sind chronologisch geordnet.

Alarm Table: Die Alarm Table zeigt nur AlarmNachrichten aus dem Control System.(Type 'event'). Sie sind erstens nach severity und dann nach timestamp sortiert. Nachrichten in der Alarm Table können quittiert werden.

Archive Table: Die Archive Table fragt die Datenbank nach Nachrichten, die in einem vom Benutzer gesetzten Zeitfenster liegen. Ein Filter kann gesetzt werden, um eine reduzierte Anzahl von Nachrichten zu erhalten.
