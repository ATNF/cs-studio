Saving Limit Changes

There are two options for saving changes made to limit values.

Save Selected Changed Limit Values

  • Highlight the row or rows containing the limit values you want saved.
  • Click the right mouse button over a highlighted cell in the column containing the changed limit value or values you would like to save. A context menu will be displayed with the Save Selected option. Slide your mouse over this option and press the left mouse button.
  • Any highligted row that contains unchanged limit values will be ignored. Only highlighted changed limits in the selected column will be displayed on the next screen. To set all the highlighted cells in the selected column to the same value see Changing Limit Values.
  • A screen for entering the e-log data will be displayed, as shown below.

  • Enter your 3 character UCAMS user id and password. Edit the title, if you'd like.
  • If the information being displayed looks correct, press the OK button to send the changed pv limit values to Channel Access and make an e-log entry.
  • If you want to terminate the transaction, press the Cancel button.

Save All Changed Limit Values

To save all the edited limit values being displayed in the table,

  • Press Ctrl+S or
  • Press the left mouse button over the File option on the menu bar, shown in the upper left corner of the screen below.

  • Select the Save option.
  • A dialog similar the the one below will be displayed,

  • Enter your 3 character UCAMS user id and password. Edit the title, if you'd like.
  • If the information being displayed looks correct, press the OK button to send the changed pv limit values to Channel Access and make an e-log entry.
  • If you want to terminate the transaction, press the Cancel button.

Once the limit values are saved, the main display will show the new limit values and the yellow cell backgrounds will no longer exist. And the Pace Editor title bar will no longer contain an *.
