Editor & Widget Palette


The editor shows the configuration of a display in the edit mode. Each display is based on a configuration file (file extension‚ *.css-sds) located in the workspace. In the edit mode there is no connection to the control system, so it is not possible to change values accidentally. To start a display in the execute mode press on the button bar and SDS will open it in a new shell. (Make sure that the display editor is saved to get your last changes).
