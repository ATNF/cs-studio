Log Table

Log Table receives messages from the JMS Servers and displays all types of messages (log, alarm, system, ...) ordered by the time stamp.


Context Menu

The context menu pops up by right click over a message / table row:

Shows all properties of the selected message in a table in an external view.
Sends all properties of the selected message via e-mail.
Delete selected message from the log view.
Delete all messages from the log view.
Copy value of property NAME to the windows clip board.
Call other applications / plug-ins available in the current CSS installation that can deal with process variables. This option is only reasonable if the property NAME of the selected message is a process variable.
