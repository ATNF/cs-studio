Adding PVs

Es gibt mehere Möglichkeiten; Prozessvariable (PVs) zum Databrowser-Plot hinzuzufügen

  • Aufrufen des Kontext-Menus des Plots, auswählen "Add PV", PV-Namen eingeben
  • Eingeben neuer oder ändern existierender PV-Namen im Data Browser Config View.
  • PV-Namen aus einer anderen CSS-Anwendung in das Plot Window oder das Config View reinziehen.
  • In anderen CSS-Anwendungen die gewünschten PV-Namen auswählen und das Kontextmenu benutzen, um sie an einen neu erzeugten Databrowser-Plot zu schicken.
