Creating an Application Launcher Configuration File

  • Right-click on the parent folder
  • Select New, Other.., section General, type File
  • As a file name, use something that ends in .app, for example Terminal.app
  • Open the file in a text editor: Right-click the launch configuration file and select Open With, Text Editor
  • Edit the file to look similar to this, see following section for details: 
    <application>
  <command>notepad.exe</command>
  <icon>icon:edit</icon>
</application>
  • Execute the launch configuration at least once viaa right-click on the file, selecting Open With, then Launch Application. From now on the double-click will also default to launching the application.
