Data Browser Waveform View

This view can display waveform samples from the current plot.

When viewing a waveform sample in the plot window, you will only see the value of the first waveform element over time. By opening the waveform view via the context menu of the plot, you can see the full waveform.

  • PV:
    Selects which PV of the current plot to view. One can currently only view one sample of one waveform PV at a time.
  • Time Slider under the plot:
    Determine which of the waveform samples of the selected PV to view.
