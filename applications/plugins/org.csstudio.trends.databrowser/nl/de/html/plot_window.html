Data Browser Plot Window

Das Plot-Window könnte als das "main"-Window des Databrowsers betrachtet werden. Seine Titelzeile enthält den Namen der zugehörigen Konfigurationsdatei oder "<not saved>" wenn eine neu erzeugte Databrowser-Instanz (noch) nicht auf eine Konfigurationsdatei gesichert wurde.

Plot Window Button Bar

Die Button Bar oben im Plot-Window bietet interaktive Pan- und Zoom-Operationen an. Sollte Die Botton-Bar nicht sichtbar sein, kann man sie mit dem Kontextmenu des Plot-Windows wieder einschalten.
  • "Up/Down", /:
    Bewegen des Charts nach oben bzw. nach unten. Wenn der Plot mehrfache Y-Achsen enthält, kann man alle Achsen oder nur eine ausgewählte Y_Achse bewegen.
  • "Y-In/Out", /:
    Zoomen der ausgewählten oder aller Y-Achsen, hinein oder heraus
  • "Auto-Zoom", :
    Zoomen der ausgewählten oder aller Y-Achsen, um die Kanaldaten des aktuellen Zeitfensters der X-Achse darzustellen.
  • "Stagger", :
    Wenn es nur eine Y-Achse gibt, ist dies das gleiche wie Auto-Zoom. Bei mehrfachen Y-Achsen werden alle Y-Achsen gezoomt, so daß sie mit ihren aktuellen Daten ohne Überlappung optimal dargestellt werden.
    Bei 2 Achsen wird die eine so autogezoomt, dass sie die obere Hälfte des Plots füllt, und die andere Achse nimmt die untere Hälfte des Plots in Anspruch.
  • "Default Zoom", :
    Benutz die Achsen-Limits, die vom PV vorgeschlagen werden. Wenn keine Vorschläge vorliegen, wird Auto-Zoom gerufen.
  • "Left/Right", /:
    Bewegen des Charts links/rechts. Moves the chart left/right.
  • "X-In/Out", / :
    Zoomen der X-Achse (Zeitachse) rein/raus.
  • "Scroll On/Off", /:
    Scroll einschalten bzw. ausschalten . Wenn Scroll eingeschaltet ist, scrollt der Plot kontinuierlich in Stirp-Chart-Art, so, daß die rechte Kante den aktuellen Zeitpunkt ("jetzt") wiedergibt.

    Die Scroll-Geschwindigkeit wird im Config View eingestellt.

    Scrolling wird bei der Durchführung manueller Pan- und Zoom-Operationen automatisch abgeschaltet, weil andernfalls das Scrolling die manuelle X-Achsen-Einstellung behindern würde.

  • "More", ...:
    Öffnen einer Dialog-Box zur Eingabe von Zeitfenstern, d.h. X-Achsen-Grenzen.

Selecting a Y Axis

Wenn ein Plot mehrfache Y-Achsen hat, können, wie oben beschrieben, die meisten Zoom- und Pan-Operationen mit allen oder mit nur einer ausgewählten Y-Achse arbeiten. Um eine Y-Achse auszuwählen, einfach anklicken. Diese Achse wird nun in einer etwas breiteren Linienbreite gezeichnet und Grid-Linien im Plot heben die ausgewählte Achse weiter hervor.

Um die Auswahl einer Y-Achse abzuschalten, anklicken der X-Achse oder einfach ausserhalb des aktuellen Plots und der Y-Achsen-Fläche klicken.

'Rubberband' Zoom

Man kann in den Plot reinzoomen mit einem "Gummiband"-Rechteck: Klicken eines Start-Punkts, Rechteck aufziehen, Taste loslassen und der Plot wird in das ausgewählte Gebiet gezoomt.

Sample Markers

Wenn man in das Plot-Window reinklickt, wird ein Marker nahe des nächstgelegenen Datenwerts plaziert. Der Kanalname, der Wert und weitere Information über diesen Datenwert werden angezeigt. Man kann mehrere Marker plazieren.

Um Marker zu löschen, wird das Kontext-Menu des Plots benutzt.

'Drop' PVs

Man kann Drag-and-Drop benutzen, um PCs zum Plot hinzuzufügen, indem man PVs in den Plot selbst oder auf eine Y-Achse "dropt". Wenn man auf den Plot dropt, wird der neue PV mit seiner eigenen neuen Y-Achse hinzugefügt. Wenn man auf eine Y-Achse dropt, wird der PV zu dieser Y-Achse hinzugefügt.

Context Menu Options

Das Kontextmenu enthält die Optionen:
  • Show/Hide Button Bar: Schaltet den button bar an oder aus.
  • Remove all Markers: Löscht alle Sample-Marker.
  • Remove selected Markers: Löscht den ausgewählten Sample-Marker. Auswahl durch Anklicken.
  • Add PV: Öffnet eine Dialog-Box zur Eingabe eines neuen PV-Names
  • Add Formula: Fügt eine Formel hinzu.
  • Config View: Öffnet die Config View.
  • Archive View: Öffnet die Archive View.
  • Sample View: Öffnet die Sample View.
  • Export View: Öffnet die Export View.
  • Open As View: Öffnet die aktuell gesicherte Databrowser-Konfigurationsdatei im Read-Only-Modus.
  • Data Browser Perspective: Wählt die Data Browser Perspective aus.
