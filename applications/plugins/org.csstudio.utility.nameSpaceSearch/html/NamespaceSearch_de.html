Name Space Search

Der Namespace Search (NsS) ermöglicht eine Suche nach Prozessvariablen in einer LDAP Struktur.

Anleitung:

Es wird Nach einer Prozessvariable gesucht. Dabei können Platzhalter in Folgender Form verwendet werden:
  • * = keins, eins oder mehrer beliebige Zeichen
  • ? = genau ein beliebiges Zeichen
Das Ergebniss wird Tabellarisch Dargestellt wobei das jeweils rechte Element das Übergeordnete Element darstellt.
Alle Elemente können durch Drag&Drop oder per Mousebutton 3 weiter verwendet werden.
