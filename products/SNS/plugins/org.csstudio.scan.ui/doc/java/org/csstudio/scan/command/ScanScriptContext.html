logData

public abstract void logData(java.lang.String device,
                             java.lang.Object data)
                      throws java.lang.Exception
Add data for a device to the log.

The data will typically be the result of a computation performed in the ScriptCommand, and it is added to the log to allow plotting or later analysis

Supported data is

  • NDArray
  • Anything that NDArray.create(Object) can read: double[], int[], ...

The data will be logged as a flat array of samples with serial numbers equal to the array index.

Parameters:
device - Device name
data - Data to log for the device

java.lang.Exception - on error
