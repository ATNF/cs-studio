Setting Preferences

Preference Page

All the preferences of BOY can be set in the preference page, which can be opened via menu: CSS->Preferences...->CSS Applications->Display->BOY.

  • Macros: The global sysmbol macros.
  • color file: The color definition file.
  • font file: The font definition file.
  • OPI GUI Refresh Cycle (ms): The refresh cycle of OPI GUI. Specify this to a longer will help to reduce CPU usage, but it may affect user's experience and may lose some swift changs if refresh cycle is longer than PV updating cycle. .
  • Automatically save file before running: The file in editing will be automatically saved before you run it.
  • No-Editing mode: OPI files are not allowed to be opened in OPI editor. They will always be opened from runtime engine.
  • Top OPIs: The OPIs which will be listed in the "Top OPIs" drop down button. If you want to add initial macros to the OPI, the format is like this: 
        "/BOY Examples/main.opi", "macro1 = hello", "macro2 = hello2" | "BOY Examples/1_2_WidgetExamples.opi"
