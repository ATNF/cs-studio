Namespace Browser

Der Name Space Browser (NSB) ermöglicht eine Suche nach Prozessvariablen in einer LDAP Struktur.

Anleitung:

Durch anklicken der einzelnen Listenelemente werden die untergeordneten Element in der nächsten Liste dargestellt. Jede Liste erhält einen Eintag "Alle" wird dieser ausgewählt so werden sämtlich untergeordneten Elemente Ausgegeben.
Die Textfelder über den Listen ermöglichen eine Filterung der Listen. Hierbei ist es erlaubt Platzhalter zu verwenden und es wird nicht nach Groß-/ Kleinschreibung unterschieden.
  • * = keins, eins oder mehrer beliebige Zeichen
  • ? = genau ein beliebiges Zeichen
Alle Elemente können durch Drag&Drop oder per Mousebutton 3 weiter verwendet werden.
