Restoring Limit Changes

  • Highlight the row or rows containing the limit values you want restored.

  • Click the right mouse button over a highlighted cell containing the changed limit value you would like to restore.

  • A context menu will be displayed with the Restore Original Value option. Slide your mouse over this option and press the left mouse button.

  • In seconds, the original value or values will be displayed on the main display screen. And the yellow background will no longer exist.
