Use external browsers

Wenn der eingebaute Web-Browser vom System unterstützt wird, nutzt das Help-System denselben, wann immer es möglich ist, und diese Option ist verfügbar. Auswählen, damit Help den externen Browser benutzt. Benutzen der "Web Browser" Preference Seite, um den gewünschten Browser auszuwählen.
