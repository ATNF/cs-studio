Data Browser

Der CSS Data Browser dient zum Ermitteln von Trends. Er kann die Werte von 'live' Prozess-Variablen (PVs) und Archivdaten in einem Stripchart-type scrolling display darstellen.

Für jeden PV versucht der Browser 'live'-Daten aus dem Kontrollsystem zu sammeln und sie über die Zeit darzustellen. Konfigurierbar sind die Größe des 'live'-Daten-Buffers und die Datennahmerate. Wenn ein Zeitbereich ausgewählt wird, der 'vor' diesem 'live'-Daten-Buffer liegt, fragt der Browser eine Archivdatenquelle nach historischen Daten. Man kannn für jeden PV ein oder mehrere Archivdatenquellen konfigurieren.
