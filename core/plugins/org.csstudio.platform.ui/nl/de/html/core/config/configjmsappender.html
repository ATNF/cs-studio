JMS Appender Settings

 

Option

Description

Default

Log level

Der Log-Level definiert die Schwelle für Log-Meldungen, die über das JMS verbreitet werden. Es werden nur Log-Meldungen verbreitet, die mindestens den gewählten Log-Level haben. Mögliche Log-Level sind : INFO, DEBUG, WARNING, ERROR, FATAL.

INFO

Pattern

Das Pattern definiert das allgemeine Layout der von CSS erzeugten Datei-Log-Meldungen. Eine Beschreibung der Pattern-Symbole findet man hier.

%d{ISO8601} %-5p [%t] %c: %m%n
