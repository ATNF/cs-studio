Data Browser Config View

Der Großteil der Konfiguration ist in dieser View dargestellt. Der Zufgriff erfolgt über das Kontextmenu des Plot Windows.
Es hat verschiedene Unterabschnitte oder "Tabs".

"Process Variables" Tab

Dies Kapitel erklärt, welche PVs im Plot gezeigt werden und wie das geht. Man kann PVs hinzufügen, indem man ihre Namen in der "<Add PV>" Zelle der "Process Variables" Tabelle eingibt. Zum Löschen von PVs: auswählen und über Kontextmenu der PV-Tabelle löschen.

Man kann die Achsenbelegung, Farbe und Linienbreite von jedem PV ändern. Die Nummer in der "Axis"-Spalte kennzeichnet die entsprechende Y-Achse 0, 1, 2 ... Um PVs auf die gleiche, erste Achse zu legen, "0" eingeben. Um sie auf zwei Achsen zu legen, "0" oder "1" für die jeweilige PV eingeben.

Die "Min" und "Max"-Einstellungen bestimmen den Y-Achsen-Bereich. Über "Type" wird eine lineare oder logarithmische Y-Achsen-Skala bestimmt. Weil diese Werte die Y-Achse verändern, betreffen sie alle PVs, die sich auf diese Achse beziehen.

Mehr Details siehe unter Display Types und Request Types.

Archive Table, Formula

Die untere Hälfte des PV Tabs ändert sich abhängig vom PV-Type. Für PVs: Es wird eine Tabelle mit Archivdatenquellen dargestellt, die benutzt werden können, um Beispiele für die ausgewählten PVs zu lesen.

Die "Re-scale for new archived data"-Auswahl bestimmt, wie mit kürzlich archivierten Daten umbegangen wird:

  • No: Kein re-scale durchführen; Achsen-Settings nicht verändern.
  • Autoscale: Wenn neue Archiv-Daten empfangen werden, dann die Y-Achsen autoskalieren.
  • Stagger: ..., stagger die Y-Achsen.

Formeln : Formeln werden angezeigt.

Change archive data sources
Um Archivdatenquellen hinzuzufügen, Drag-and-Drop ein Archiv vom Archive View auf einen PV. Man kann sie auf das Plot-Window droppen, welches die Quelle zu allen PVs im Plot hinzufügt, oder auf einen ausgewählten PV in der Konfigurations-View.
Im Kontextmenu der "Archives used for ..."-Tabelle kann man die Reihenfolge der Archive für einen PV neu ordnen.
Um eine Archivdatenquelle von einem PV zu löschen, PV im Konfigurations-View auswählen, Archivdatenquelle in der "Archives used for ..."-Tabelle auswählen und über das Kontextmenu löschen.

"Live Data Config" Tab

Dies Kapitel beschreibt die Einstellungen, wie oft 'life'-Daten für die PVs genommen werden und wie oft der Plot neu gezeichnet wird. Man kann beispielsweise einen PV alle 2 Sekunden auslesen und den Plot alle 4 Sekunden neu zeichnen. Man beachte, dass Neuzeichnen viel CPU-Zeit verbraucht. Man spart CPU-Zeit, wenn man die Neuzeichnen-Rate kleiner und die Update-Rate größer dreht.

"Time Config" Tab

Gestattet Eingabe von Start- und End-Zeitpunkt.

Für ein "scrolling" Display gibt man relative Zeiten mit Endwert "now" ein. Beispiel: Startzeit = "-30 minutes" und Endzeit "now". Man benutzt besser den "start/end time dialog", der über die "..."-Knöpfe erreichbar ist, wenn die start/end Time Syntax nicht klar ist.
