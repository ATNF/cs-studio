Overview

This is far from done! It is my playground for trying things while reading the "eclipse - Building Commercial-Quality Plug-ins" book.

Basic Idea

The PV Table provides a tabular view of PV names and their current value. One can start and stop live value updates.

In addition, one can take a "snapshot" of current values, which gets saved when saving the PV table document. The PV table display indicates current values that differ from the "snapshot" values in red.

One can configure the update rate as well as the threshold for indicating differences between the current values and those from the snapshot.
