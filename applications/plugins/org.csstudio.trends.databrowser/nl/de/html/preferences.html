Data Browser Preferences

Voreinstellungen werden über das Menu

CSS/Preferences

konfiguriert, dann nach

CSS Applications/Trends/Data Browser

gehen.

Options

Default to Autoscale for new PV

Sollen neu hinzugefügte PVs "Autoscale" benutzen ?

Show Request Types

Vorzugsweise fordert der Datenbrowser-Plot Daten an, die für das Plotten optimiert sind und stellt automatisch Rohdaten dar, wenn man weit genug reinzoomt.

Wer jedoch lieber Speicher verbraten und die Anwendung abstürzen möchte, hat die Option, Benutzer-einstellbare Datentypen, incl. Rohdaten, einzustellen.

Plot Bins

Anzahl von optimierten Samples, die der Datenbrowser abfragt. Ein Wert um die 200 ist ein guter Kompromiss: Man sieht, wie die Daten aussehen (Min, Max, Mittelwert), ohne zu viel Speicher und Netzwerk-Resourcen zu verbrauchen.

Default Start/End Time

Start- und End-Zeiten, die für die Plots benutzt werden.

Archive Server URLs

Voreingestellte Archiv-Server-URLs werden im Archive View gezeigt.

Default Archives for new PVs

Voreingestellte Archiv-Server-URLs und Tasten für die PVs. Im Datenbrowser-Editor kann man sie ändern oder neue aus Archive View hinzufügen.
