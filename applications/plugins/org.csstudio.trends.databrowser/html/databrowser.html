Data Browser

The CSS Data Browser is a trending tool. It can display the values of 'live' Process Variables (PVs) in a Stripchart-type scrolling display, as well as archived data.

For each PV, it will try to collect 'live' samples from the control system and plot them over time. The size of this 'live' sample buffer is configurable, as is the sample period. When selecting a time range that lies 'before' this live sample buffer, the tool will query an archive data source for historic samples. You can configure one or more archive data source for each PV.
