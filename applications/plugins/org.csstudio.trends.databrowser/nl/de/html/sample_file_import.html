Data Browser Sample File Import

Mit dem Datendatei-Import kann man Daten aus einer externen Datei im Datenbrowser darstellen. Wenn mehr als 100000 Werte vorliegen, kann es Verzögerungen bei der Grafik fü '*zoom', 'shift', ... geben, weil der Datenbrowser dafür gebaut ist, Daten von einem Archiv-Server zu holen. Der Datendatei-Import ist nicht in allen CSS-Versionen verfügbar.

Die Datendatei-Import-Aktion ist im Kontextmenu (Rechtsklick) von Config View verfügbar.

CSV Files

Im Dateibrowser kann man die Datendatei zum Importieren auswählen. Um zwischen den verschiedenen Formaten der Datendateien zu unterscheiden, benutzt der Datendatei-Import die Dateierweiterung *.cvs-<scope-id> . Zur Zeit werden diese Formate unterstützt

Import Settings

Nach Auswahl der Datendatei zeigt der nächste Dialog die Kanäle in der Datei mit der verfügbaren Information.

Choose the records to display

In diesem Abschnitt steht eine Liste aller Kanäle in der Datei mit Namen, Start- und End-Zeit der Messung, Anzahl der zu importierenden Daten, Gesamtanzahl von Daten in der Datei, Einheit für Kanal und eine Formel zur Nachberechnung der Daten. Feldnamen und Einheiten sind editierbar; man kann sie vor dem Import ändern. Die Checkbox am Anfang jeder Zeile dient zur Kennzeichnung der Kanäle, die importiert werden sollen.

Time range for graph

Das Start/End-Feld zeigt die aktuell ausgewählte Start- und End-Zeiten. Wenn kein Datenbrowser-Editor geöffnet ist, wird die Default-Zeit aus der Datendatei gelesen. Wenn ein Editor geöffnet ist, wird die Default-Zeit aus dem Zeitbereich des Graphen gelesen. Mit dem Knopf "Time" kann man das Zeitfenster manuell verändern. Der Knopf "from Graph" liest das Zeitfenster aus dem aktiven Editor und der Knopf "from File" aus der Datendatei.Die Datenanzahl im vorigen Abschnitt wird automatisch aktualisiert.

Factor

Besonders bei großen Dateien ist es wichtig, die Anzahl der zu übertragenden Daten zu reduzieren. Bei mehr als 100.000 Daten wird der Durchsatz des Datenbrowsers langsam. Der Datenimport liest jedes Datum.

Time offset

Hinzufügen eines Zeit-Offset zur Zeit in der Datei. Wenn man Kanäle von verschiedenen Quellen mit verschiedenen Zeitskalen importieren möchte, kann man die Datei-Zeit einstellen, um die Daten zu synchronisieren.
