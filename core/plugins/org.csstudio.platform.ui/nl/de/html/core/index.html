Overview

Welcome to CSS!

CSS heisst Control System Studio. CSS ist eine allgemeine Plattform für verschiedenartige Control System Anwendungen. Für Benutzer dient sie als ein Einsprungspunkt zum Start von Anwendungen, die Control Systeme als 'back ends' benutzen. Für Entwickler ist es eine Plattform, auf der portierbare Anwendungen für verschiedene Control Systeme geschrieben werden.

Dies Manual ist für Benutzer von CSS und seinen Anwendungen gedacht.

Historical Background

Im Jahre 2005 hatte Matthias Clausen von DESY (Hamburg) die Idee, ein neues Controlling Software System zu bauen, um existierende Software-Unzulänglichkeiten zu beseitigen. Heutige Anwendungen ...

 

  • ... basieren auf verschiedensten Technologien.
  • ... unterscheiden sich in 'look and feel'.
  • ... lassen eine einheitliche Steuerung vermissen.
  • ... benötigen spezielle Betriebssysteme oder Backends.
  • ... sind nicht für Kollaborationen geeignet (z.B. Datenaustausch zwischen Anwendungen).

Technical Approach

Wir haben uns entschieden, die neue CSS-Software als Eclipse Rich Client Anwendung zu implementieren. Die Plugin-Architektur von Eclipse macht es relativ einfach, das System mit eigenen Features und Funktionen zu erweitern. Weil nicht alle Entwickler firm im Entwickeln von Eclipse-Plugins sind, werden wir ausführliche Dokumentation und code templates bereitstellen, um die Entwicklung neuer Plugins zu beschleunigen.

 

Wer sich mit der Entwicklung von Plugins für CSS beschäftigen möchte, sollte sich auch die CSS Homepage ansehen.
