Console Appender Settings

 

Option

Description

Default

Log level

Der 'log level' definiert die Schwelle für Log-Meldungen, die auf der Konsole ausgegeben werden. Es werden nur Log-Meldungen ausgegeben, die mindestens den eingestellten log level haben. Mögliche 'log level' sind: INFO, DEBUG, WARNING, ERROR, FATAL.

INFO

Pattern

Dieses Bitmuster definiert das allgemeine Layout von CSS-Konsolenmeldungen. Eine Beschreibung findet man hier

%d{ISO8601} %-5p [%t] %c: %m%n
