Data Browser Sample View

Diese View erlaubt die Auswahl eines PVs aus dem aktiven Plot und zeigt seine Daten in tabellarischer Form. Man kann sie über das Kontextmenu von plot window öffnen.

Dies Kommando ist nützlich, wenn man exakte numerische Werte und timestamps eines PV benötigt. Der Marker von plot Window kann diese Info für ein paar Daten liefern, aber ein Tabellen-View ist manchmal nützlicher.

Man kann auch Export View benutzen, um so eine Datenliste auf eine Datei zu sichern.

Man beachte, dass die Daten-View nicht automatisch refresht wird, sobald neue Daten von einer Archiv-Anfrage oder, als 'live'-Daten, vom Kontrollsystem ankommen, weil die dann notwendigen Tabellenaktualisierungen zu aufwändig wären. Stattdessen muss man die Tabellen-Aktualisierung mithilfe des entsprechenden "Refresh"-Knopfes manuell triggern.
