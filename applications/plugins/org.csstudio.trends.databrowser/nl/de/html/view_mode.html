Data Browser View Mode

Wenn man den Datenbrowser über das CSS-Menu, aus dem Kontextmenu eines PV oder durch Öffnen einer existierenden Datenbrowser-Konfigurationsdatei startet, läuft er im edit-Modus.
Wie der Name sagt, erlaubt einem dieser Modus zu editieren, wie z.B. durch Verändern der Datenbrowser-Konfiguration mittels Hinzufügen von PVs mit Drag-und-Drop, durch Öffnen der zugehörigen Config View und Archive View.

Ansonsten kann man das Kontextmenu einer Datenbrowser-Konfigurationsdatei oder eines ausgewählten PVs verwenden, um eine Datenbrowser-Instanz im view-Modus zu öffnen.
Das ist ein Read-only-Modus mit eingeschränker Funktionalität. Es werden nur Änderungen unterstützt, die vom Button Bar des Plot-Windows angeboten werden; der Button-Bar ist anfänglich unsichtbar, er muss über das Kontextmenu des View-Mode-Windows geöffnet werden.
Der einzige Vorteil: View-Windowss bieten einige zusätzliche Positionierungsoptionen, u.a. auch einen "detached"-Modus.
