For the beginning there is a project with some example SNL files. To install the project in the CSS workspace please select Menu 'CSS' → 'Untilities' → 'Install Sample Projects'.

Set the SNL perspective: Menu 'CSS' → 'Editors' → 'SNL'. Open the example files with the extension *.st in project 'SNLSampleProject', folder 'source' with double click in the SNL Editor.
