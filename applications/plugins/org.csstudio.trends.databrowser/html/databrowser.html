Overview

The CSS Data Browser is a trending tool. It can display the values of 'live' Process Variables (PVs) in a Stripchart-type scrolling display, as well as archived data.

Components

Main "Plot" window

"Config" view

"Archive" view

"Sample" view

"Export" view

Usage

Add PVs, see what happens.
