Data Browser Archive View

Diese View gestattet browsen von auf einem Archiv-Server verfügbaren Archiven, suchen nach Prozessvariablen-Namen und ziehen von Namen aus Suchergebnissen in den Plot. Man kann die View über das Kontextmenu des plot windows öffnen.

URL: ..

Auswählen der Archivdaten-Server-URL. Der "Info"-Knopf weiss mehr über den Server.

Man kann neue URLs manuell eingeben. Die Initial-URL ist in "Einstellungen" gesetzt. Fragen Sie Ihren Site-Archiv-Administrator nach gültigen URLs.

Archive/Key/Description Table

Eine Liste von Archiven auf der gewählten URL. Man kann ein oder mehrere Archive auswählen, um PVs zu suchen. Man kann auch Archivnamen in das

  • Plot Window oder das
  • Config View

des Databrowsers reinziehen, was den Databrowser veranlasst, diese Archive für sämtliche oder ausgewählte PVs zu benutzen. Einzelheiten zu "drop" siehe "plot window" and "config view" Dokumente.

Pattern:...

Man kann ein Suchmuster für Prozessvariablen-Namen eingeben, um in der Archivtabelle ausgewählte Archive zu durchsuchen. Ist kein Archiv ausgewählt, werden alle Archive durchsucht. Betätigen von "<Return>" oder des "Search"-Knopfes startet die Suche.

Das Muster ist ein regulärer Ausdruck:

Name/Archive/Start/End Table

Diese Tabelle listet die Ergebnisse einer Suche auf. Für jeden Namen wird angegeben, in welchem Archiv er gefunden wurde. Wenn möglich, wird auch Start- und End-Zeit des Messwert-Bereichs angegeben, der im Archiv verfügbar ist.

Man kann einen oder mehrere PVs mit ihrer Archivinformation in das Databrowser-Plot-Fenster bzw. das Konfigurations-View reinziehen oder man kann das Kontextmenu der Tabelle benutzen, um ausgewählte PVs an andere CSS-Anwendungen zu schicken.
